# 名词格的变化

你好！我是你的专属德语老师。你提出了一个非常有代表性的德语学习疑问。你观察得很敏锐：在德语中，格（Kasus）的变化确实**主要**是通过冠词（如 der/den/dem/des）来体现的。

但是，你的结论“名词完全不用根据格变化”**是不准确的**。为了让你的德语达到扎实的 B 1 水平，我们必须纠正这个误区。事实上，德语中有**三种非常重要的情况**，名词本身的词尾必须跟着发生变化。

让我们来详细梳理一下这些例外情况：

### 1. 阳性和中性名词的单数第二格 (Genitiv Singular)

对于大多数阳性（Maskulinum）和中性（Neutrum）名词，在单数第二格时，名词词尾必须加上 **-s** 或 **-es**。

- **规则**：单音节名词通常加 **-es**，多音节名词通常加 **-s**。
- **示例**：
    - 第一格：_der Mann_ (男人) -> 第二格：_das Auto **des Mannes**_ (男人的车)
    - 第一格：_das Kind_ (孩子) -> 第二格：_das Spielzeug **des Kindes**_ (孩子的玩具)
    - 第一格：_der Vater_ (父亲) -> 第二格：_der Computer **des Vaters**_ (父亲的电脑)

### 2. 复数第三格 (Dativ Plural)

这是 B 1 考试中极其容易丢分的地方！在复数第三格时，如果名词的复数词尾不是 **-n** 或 **-s**，那么必须在词尾强行加上一个 **-n**。

- **示例**：
    - 第一格/第四格复数：_die Kinder_ (孩子们) -> 第三格复数：_mit den Kinder**n**_ (和孩子们一起)
    - 第一格/第四格复数：_die Freunde_ (朋友们) -> 第三格复数：_ich helfe den Freunde**n**_ (我帮助朋友们)
    - （_例外_：如果是 _die Autos_，第三格依然是 _den Autos_，因为已经是 -s 结尾。）

### 3. 阳性弱变化名词 (N-Deklination)

[[阳性弱变化名词#阳性弱变化名词]]

这是 B 1 阶段最核心、也是最高级的语法点之一。有一类特殊的阳性名词（通常表示人、动物，或者以 -e, -ent, -ant, -ist 等外来词后缀结尾），它们在除了单数第一格（Nominativ）以外的**所有格**（第四格 Akkusativ、第三格 Dativ、第二格 Genitiv）中，名词词尾都必须加上 **-n** 或 **-en**。

- **示例**：
    - _der Student_ (大学生) -> 第四格：_Ich frage den Student**en**._ (我问这个大学生。)
    - _der Junge_ (男孩) -> 第三格：_Das Buch gehört dem Junge**n**._ (这本书属于这个男孩。)
    - _der Herr_ (先生) -> 第二格：_die Tasche des Herr**n**_ (这位先生的包)
