# 高频动词

德语动词的数量非常庞大（常用的约有 3,000 到 5,000 个，总数更是数以万计），且根据时态、人称变位极其复杂。**一次性返回所有动词是不现实的，也不利于记忆。**


但我可以为你精选**德语中最高频、最核心的 10 个基础动词**。掌握这 10 个词，你就掌握了德语语法的半壁江山（包括助动词、情态动词的基础）。

以下是严格按照结构解析的德语核心动词列表：

---

### sein #ak

- 🗣️ **谐音发音**：**zain** (拼音：zài-n)。
    - **发音逻辑**：元音组合 **ei** 永远发类似拼音的 "ai" (爱)；辅音 **s** 在元音前发浊音 /z/ (类似里的 z)；**n** 是鼻音收尾。重音在整体。
- ⭐ **中文翻译**：是；存在 (相当于英语的 be)。
- **词根词缀**：不规则动词，无明显规则词根。它是德语中最不规则的动词（变位：ich bin, du bist...）。
- 🧠 **助记**：
    - **联想**：这个词的发音像“**在**”(zain -> zai)。
    - **记忆口诀**：“我**在**，故我**是** (sein)。”

### haben #ak

- 🗣️ **谐音发音**：**哈-ben** (拼音：hā-ben)。
    - **发音逻辑**：**h** 是送气音，像喝水的“喝”；**a** 是长音 /a:/；**b** 发 /b/；词尾 **-en** 弱读，发音部位很轻，接近“本”但不卷舌。
- ⭐ **中文翻译**：有；拥有 (相当于英语的 have)。
- **词根词缀**：词根 **hab-** (拥有) + 后缀 **-en** (动词不定式后缀)。
- 🧠 **助记**：
    - **联想**：发音像“**哈**”。
    - **情景**：我**有**很多钱，所以我**哈**哈大笑。

### werden #ak

- 🗣️ **谐音发音**：**V-ear-den** (类似：韦尔-登)。
    - **发音逻辑**：**w** 在德语里永远发咬唇音 /v/ (英语的 v)；**er** 在非词尾时发 /ɛr/ (开口较大的 e + 喉咙深处的小舌音)；词尾 **-en** 弱读。
- ⭐ **中文翻译**：变成；成为；(作为助动词构成将来时/被动语态)。
- **词根词缀**：词根 **werd-** (转变) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音有点像 English 的 "Word" (单词)。
    - **故事**：那个孩子从小背 **Word** (单词)，长大后**变成** (werden) 了一名作家。

### machen #ak

- 🗣️ **谐音发音**：**妈-hen** (拼音：mā-hen)。
    - **发音逻辑**：**a** 发长音 /a:/；**ch** 在 a/o/u 之后发喉音 /x/ (像深喉清理喉咙的声音，或“喝”的加重版)；词尾 **-en** 弱读。
- ⭐ **中文翻译**：做；干；制造 (相当于英语的 make/do)。
- **词根词缀**：词根 **mach-** (制作) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音极像中文的“**马**”。
    - **故事**：这匹**马**很勤劳，什么活都肯**做** (machen)。

### gehen #ak

- 🗣️ **谐音发音**：**给-en** (拼音：géi-en)。
    - **发音逻辑**：**g** 发 /g/；**eh** 中的 **h** 是“延长音符号”(Dehnungs-h)，不发音，只把前面的 **e** 拉长并变成紧音 /e:/ (嘴唇向两侧咧开)；词尾 **-en** 弱读。
- ⭐ **中文翻译**：走；去；(指身体移动)。
- **词根词缀**：词根 **geh-** (走) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音像“**给**”。
    - **故事**：你**走**吧，我把钱**给** (gehen) 你。

### kommen #ak

- 🗣️ **谐音发音**：**考-men** (拼音：kǎo-men)。
    - **发音逻辑**：**k** 发送气音 /k/；**o** 发短音 /ɔ/ (嘴巴张圆，短促)；双写 **mm** 表示前面的元音发短音；词尾 **-en** 弱读。
- ⭐ **中文翻译**：来；来到。
- **词根词缀**：词根 **komm-** (来) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音像“**考**”。
    - **故事**：老师**来** (kommen) 了，马上就要**考**试了。

### sagen #ak

- 🗣️ **谐音发音**：**扎-gen** (拼音：zà-gen)。
    - **发音逻辑**：**s** 在元音前发浊音 /z/ (拼音 z)；**a** 是长音；**g** 发 /g/；词尾 **-en** 弱读。
- ⭐ **中文翻译**：说；告诉。
- **词根词缀**：词根 **sag-** (说) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音像“**砸**”。
    - **故事**：他**说** (sagen) 谎话，我很想**砸**他。

### sehen #ak

- 🗣️ **谐音发音**：**z-ei-en** (拼音：zēi-en)。
    - **发音逻辑**：**s** 发浊音 /z/；**eh** 中的 **h** 不发音，将 **e** 拉长发紧音 /e:/；词尾 **-en** 弱读。
- ⭐ **中文翻译**：看；看见。
- **词根词缀**：词根 **seh-** (看) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音接近 English 的 "See" 加上尾音。
    - **故事**：用眼睛 (See) 去 **sehen** (看)。

### essen #ak

- 🗣️ **谐音发音**：**A-sen** (拼音：āi-sen)。
    - **发音逻辑**：**e** 发短音 /ɛ/ (开口度大，像“爱”的开口音)；双写 **ss** 表示前面的元音短促，**ss** 发清辅音 /s/；词尾 **-en** 弱读。
- ⭐ **中文翻译**：吃。
- **词根词缀**：词根 **ess-** (进食) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：发音像“**爱森**”。
    - **故事**：**爱森**斯坦 (爱因斯坦的弟弟) 很喜欢**吃** (essen) 东西。
<!--ID: 1771319860662-->
### trinken #ak

- 🗣️ **谐音发音**：**特-rin-ken** (拼音：t-rīn-ken)。
    - **发音逻辑**：**t** 发送气音；**r** 发小舌颤音 (如果发不出，可以用喉咙底部的哈气声代替)；**i** 发短音；**nk** 组合发 /ŋk/ (带鼻音，像 English 的 drink)。
- ⭐ **中文翻译**：喝。
- **词根词缀**：词根 **trink-** (饮) + 后缀 **-en**。
- 🧠 **助记**：
    - **联想**：和英语 **Drink** 几乎一模一样。
    - **故事**：Drink 就是 **trinken**，把 D 换成 T 就能**喝**了。

### 📚 语法补充：德语动词的基础规则

针对初学者，这几个动词的**人称变位**是最重要的语法点。德语动词不像中文一样不变，而是随主语变化。


1. **规则变位逻辑 (以 machen 为例)**：
    
    - 绝大多数动词去掉词尾 `-en`，加上以下词尾：
    - 我 (ich) -> **-e** (mache)
    - 你 (du) -> **-st** (machst)
    - 他/她/它 (er/sie/es) -> **-t** (macht)
    - 我们 (wir) -> **-en** (machen - 还原)
    - 你们 (ihr) -> **-t** (macht)
    - 您/他们 (Sie/sie) -> **-en** (machen - 还原)
        
2. **必须死记硬背的特殊变位 (Sein - 是)**：
    
    - Ich **bin** (我是)
    - Du **bist** (你是)
    - Er/Sie/Es **ist** (他是)
    - Wir **sind** (我们是)
    - Ihr **seid** (你们是)
    - Sie **sind** (您是)
