# 德语四格 (Kasus) 核心语法笔记

## 1. Nominativ (第一格 / N)

**定位：** 最基础的格，名词词性的原始形态。
ID: 1774612231195


### 📌 核心功能

1.  **充当句子主语** (最主要功能)
    * *例*：**Der Tisch** ist neu. (桌子作主语)
    * *例*：An der Wand steht **der Tisch**. (语序改变，功能不变，仍是主语)
2.  **充当表语** (跟在系动词之后，主语的另一面)
    * **“一格动词” (系动词)**：`sein` (是/be), `werden` (成为), `bleiben` (仍然)
    * *例*：Das ist **mein Tisch**. (Tisch 是 der，第一格物主代词用 mein)
    * *例*：Sie bleibt **eine Lehrerin**.
ID: 1774612231199


### 📌 人称代词的对应与指代 (绝对跟着词性走)

*德语的第三人称单数 (er/es/sie) 既可指人，也可代物。代物时必须严格对应名词的原始词性！*
ID: 1774612231203


* `der` ➡️ `er` (阳性) | *例*：Der Tisch ist neu. **Er** ist auch billig.
* `das` ➡️ `es` (中性) | *例*：Das Mädchen ist sehr nett. **Es** ist drei Jahre alt.
* `die` ➡️ `sie` (阴性) | *例*：Ich habe eine Maus. **Sie** ist klein.

### 📌 词尾派生规律

记住 `der/das/die`，就等于记住了一大串第一格的词尾规律：
ID: 1774612231207


| 定冠词 (N) | 疑问代词 (which) | 指示代词 (these) | 词尾特征 |
| :--- | :--- | :--- | :--- |
| **der** | welch**er** | dies**er** | **-er** |
| **das** | welch**es** | dies**es** | **-es** |
| **die** | welch**e** | dies**e** | **-e** |

## 2. Akkusativ (第四格 / A)

**定位：** 德语中运用最广泛的格，与第一格高度相似，仅阳性发生变化。
ID: 1774612231210


### 📌 核心功能

1.  **充当直接宾语** (跟在“四格动词 / 及物动词”后)
    * *例*：Ich esse **den Apfel**. / **Den Apfel** esse ich. (Apfel 是 der，作宾语变为 den)
2.  **双宾动词中的直接宾语** (口诀：**人三物四**)
    * 当动词（如 `geben`）后接两个宾语时，**人**用第三格 (D)，**物**用第四格 (A)。
    * *例*：Er gibt mir **ein Buch**. (mir 是人的 D，ein Buch 是物的 A)
3.  **作时间状语** (单独存在)
    * *例*：**Jedes Jahr** besuche ich Herrn Wang. (每年)
    * *例*：**jeden Samstag** (每周六), **jedes Wochenende** (每周末)
4.  **特定介词硬性要求**
    * `gegen`, `ohne`, `für`, `um`

### 📌 N ➡️ A 的变形规律 (极简)

**规律**：只有阳性 (m.) 发生变化（词尾变为 **-en**），其余中性 (n.)、阴性 (f.)、复数 (pl.) 均与第一格保持一致。
ID: 1774612231215


| 词类 | 阳性 (m.) | 中性 (n.) | 阴性 (f.) | 复数 (pl.) |
| :--- | :--- | :--- | :--- | :--- |
| **定冠词 (N)** | der | das | die | die |
| **定冠词 (A)** | **den** | das | die | die |
| 不定冠词 (A) | **einen** | ein | eine | / |
| 否定词 (A) | **keinen** | kein | keine | keine |
| 物主代词 (A) | **meinen** | mein | meine | meine |
| 疑问代词 (A) | **welchen** | welches | welche | welche |
| 指示代词 (A) | **diesen** | dieses | diese | diese |
| 不定代词 (A) | **jeden** | jedes | jede | / |

## 3. Dativ (第三格 / D)

**定位：** 德语 DNA 的重要组成部分，常用于特定动词、双宾语的人、以及表示静态位置。
ID: 1774612231219


### 📌 核心功能

1.  **特定动词硬性要求** (考试重难点)
    * `helfen` (口诀：**帮三**) ➡️ *例*：Ich helfe **der Frau**. (绝不能用 die Frau，避免“不三不四”)
    * `antworten` (口诀：**问四答三**) ➡️ `fragen` 接 A，`antworten` 接 D。*例*：Er antwortet **mir** nicht.
2.  **双宾动词中的间接宾语** (口诀：**人三物四**)
    * *例*：Er gibt **der Frau** ein Buch. (der Frau 即为充当间接宾语的第三格)
3.  **特定介词硬性要求**
    * `zu`, `aus`, `von`, `mit`, `nach`, `gegenüber`, `seit`, `entsprechend`, `laut`, `gemäß`。
    * 特殊情况：`in` 表示时间时一定接第三格；`entlang` (沿着) 的格取决于它的位置。
4.  **表示“对...... 来说”**
    * *例*：Es ist **mir** egal. (对我来说无所谓，只能用 mir)
5.  **介词的“静三动四”原则 (极其重要！)**
    * 适用于 **9 大双格介词**：`auf`, `unter`, `über`, `neben`, `an`, `in`, `zwischen`, `vor`, `hinter`。
    * **静三 (Dativ)**：表示“静态地点” (在... 地方 / Where)。
        * *例*：Ich bin **am Meer**. (我在海边) / Er sitzt **am Fenster**.
    * **动四 (Akkusativ)**：表示“动态方向” (去... 地方 / To where) 或 “抽象状态的改变”。
        * *具体方向*：Ich fahre **ans Meer**. (我去海边) / Er setzt sich **ans Fenster**.
        * *抽象状态*：Er übersetzt den Text **ins Spanisch**. (译成西班牙语，状态改变)

## 4. Genitiv (第二格 / G)

**定位：** 相对较少用，主要用于表示所属关系及少数特定搭配。
ID: 1774612231225


### 📌 核心功能

1.  **表示所属关系 (“A 的 B” 结构)**
    * 规则：中文的“A 的 B”在德语中翻成 **“B A”** 结构，其中 A (所有者) 后置且变为第二格。前提是 A 和 B 都是名词。
    * *例*：“我姐姐的男朋友” ➡️ der Freund **meiner Schwester**. (无论 Freund 在句中充当什么格，meiner Schwester 永远是第二格)。
2.  **少数动词硬性要求**
    * `bedürfen` (需要) ➡️ *例*：Ich bedarf **eines Buchs**. (相比之下，同义词 `brauchen` 只需接 Akkusativ：Ich brauche ein Buch.)
3.  **特定介词硬性要求**
    * `einschließlich`, `ausschließlich`, `anlässlich`, `wegen`, `trotz`, `innerhalb`, `außerhalb`。
4.  **固定用法**
    * *例*：**eines Tages** (有一天)。

## 5. 附录：德语“生命表” (核心变化全览)

### 👑 德语的 DNA：定冠词四格变形表 (N-G-D-A)

> **记忆规律**：
> * **G (第二格)**：阳性与中性一致 (`des`)，阴性与复数一致 (`der`)。
> * **D (第三格)**：阳性与中性一致 (`dem`)。
> * **万能钥匙**：所有乱七八糟的词（ein/kein/物主代词/welch-/dies-/jed-/solch-等）的词尾，统统由此表推断得出！

| 格 (Kasus) | 阳性 (m.) | 中性 (n.) | 阴性 (f.) | 复数 (pl.) |
| :--- | :--- | :--- | :--- | :--- |
| **Nominativ (一格)** | **der** | **das** | **die** | **die** |
| **Genitiv (二格)** | **des** | **des** | **der** | **der** |
| **Dativ (三格)** | **dem** | **dem** | **der** | **den** |
| **Akkusativ (四格)** | **den** | **das** | **die** | **die** |
