# 未命名

- [ ] 普通待办 (未完成)
- [x] 已完成
- [/] 进行中 (In Progress)
- [-] 已取消 / 删除线 (Canceled)
- [>] 已推迟 / 延期 (Deferred)
- [<] 提前 / 计划中 (Scheduling)
- [?] 疑问 / 待确认 (Question)
- [!] 重要 / 警告 (Important)
- [*] 星标 / 关注 (Star)
- ["] 引用 (Quote)
- [l] 地点 / 坐标 (Location)
- [b] 书签 (Bookmark)
- [i] 信息 / 提示 (Info)
- [S] 金额 / 赞助 (Amount)
- [I] 灵感 / 想法 (Idea)
- [p] 优点 (Pro)
- [c] 缺点 (Con)
- [f] 火焰 / 热点 (Fire)
- [k] 钥匙 / 关键 (Key)
- [w] 胜利 / 成功 (Win)
- [u] 向上 (Up)
- [d] 向下 (Down)
